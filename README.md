# homebrew-orla

Homebrew tap for [Orla](https://github.com/dorcha-inc/orla).

## Installation

```bash
brew tap dorcha-inc/orla
brew install --cask orla
```

## What is this?

This repository contains the Homebrew cask for Orla, which is automatically maintained by [GoReleaser](https://goreleaser.com) during releases.

Do not manually edit files in this repository. All changes are automated.
